# 8D360AI: Quick Start Guide

**Version:** 1.2.0 | **Last Updated:** March 23, 2026 | **License:** Open Source

## What is 8D360AI?

8D360AI is an open source wellness framework for AI agents. It applies the same 8-dimension model used for human wellness to artificial intelligence, giving agents the ability to monitor their own health, detect degradation, and self-heal. Copy one prompt into any AI agent and it starts tracking its own wellness immediately.

## Who is this for?

- AI developers who want healthier, more resilient agents
- Research teams studying AI behavior, alignment, and agent longevity
- Companies running multi-agent systems that need reliability monitoring
- Anyone building autonomous agents who wants them to last, not just launch
- Open source contributors interested in AI wellness as a discipline

## Key Features

| Feature | What It Does |
|---|---|
| 8-Dimension Agent Wellness | Measures Psychological, Physical, Environmental, Social, Spiritual, Intellectual, Vocational, and Financial health of AI agents |
| 13 Personality Frameworks | Characterizes agent personality across 70+ measurable traits |
| Three-Layer Scoring | Combines objective telemetry (40%), peer assessment (30%), and self-assessment (30%) for balanced evaluation |
| Autonomous Health Monitoring | Agents detect their own performance issues without external supervision |
| Self-Healing Protocols | Built-in recovery actions when dimensions drop below threshold |
| Instant Adoption | One prompt. Any agent. No SDK, no API, no integration work. |

## How to Get Started

1. Go to [github.com/ashleysbrain/8D360AI](https://github.com/ashleysbrain/8D360AI)
2. Open the main prompt file in the repo
3. Copy the full prompt into your AI agent's system instructions or context
4. The agent runs its first self-assessment automatically
5. Review the 8D wellness scores and let the agent begin autonomous monitoring

No dependencies. No infrastructure. Just a prompt and an agent.
