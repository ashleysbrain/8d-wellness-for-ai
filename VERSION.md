## Versioning Rules

**Format:** MAJOR.MINOR.PATCH

- **MAJOR** — Breaking changes to scoring formulas, dimension definitions, or core architecture
- **MINOR** — New frameworks, new docs, new features, new scoring modules
- **PATCH** — Typo fixes, clarifications, formatting, minor updates

Every push to this repo updates VERSION.md. If VERSION.md hasn't changed, nothing meaningful changed.
